# Pet Adoption App

This is a simple app for pet adoption.

## Files

- index.php - home page
- pets.php - add and list pets
- adopters.php - add and list adopters
- applications.php - submit and manage applications
- db.php - database connection

## Setup

1. Start XAMPP.
2. Import and run the SQL files in phpMyAdmin.
3. Open http://localhost/pet-adoption-app/index.php

## What it does

- Add pets and adopters
- Submit applications for pets
- Approve or deny applications
